# Proyecto Android + Spring Boot + MySQL + Firebase

Este repositorio contiene un proyecto full-stack con:

- **Android**: aplicación móvil (carpeta `android-app/`).
- **Spring Boot**: API backend en Java (carpeta `backend/`).
- **MySQL**: base de datos relacional.
- **Firebase**: servicios en la nube (por ejemplo Auth, Cloud Messaging).

## Estructura inicial

- `backend/`: código del backend con Spring Boot.
- `android-app/`: proyecto de la app Android.
- `infra/`: archivos de infraestructura (MySQL, Firebase, etc.).

## Próximos pasos

- Inicializar el proyecto de Spring Boot dentro de `backend/`.
- Crear el proyecto de Android Studio dentro de `android-app/`.
- Configurar la base de datos MySQL y la integración con Firebase.
